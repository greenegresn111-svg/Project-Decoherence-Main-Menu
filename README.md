# Project Decoherenece

## Project Description
This is a main menu mockup for my personal video game project.

## Intended User or Player
This userface is desined for people of all ages who play video games.

## User / Player Goal
The user should be able to start a new game, load a saved game, edit the game settigns, and quit the game.

## Inspiration Interfaces
List the inspiration interfaces you used and briefly explain what you learned from them.

## Design Choices
- Layout: Main Menu > New Game, Load Game, Settings, Quit
- Color: Purple/Gray
- Typography: "Orbitron", "Segoe UI", Tahoma, Geneva, Verdana, sans-serif
- Visual hierarchy: Title > Background, New Game > Load Game > Settigns > Quit
- Feedback states: Button Hover-state 
- Responsive design: Website scales with screen size
